Our Programs

Explore our flagship programs designed to prepare you for careers in technology.

Computer Science

Overview: Build strong foundations in programming, algorithms, systems, and software engineering. Our Computer Science program teaches theoretical concepts and practical skills used to design and develop modern software.

Core Modules: Data Structures & Algorithms, Operating Systems, Databases, Software Engineering, Computer Networks, Web Development, Programming Languages.

Duration: 3 years (full-time) / flexible part-time options available.

Career Paths: Software Developer, Systems Engineer, DevOps Engineer, Backend/Frontend Engineer, Technical Lead.

Enquire / Apply

Data Science

Overview: Learn to collect, process, analyze and visualise data to make data-driven decisions. The program covers statistics, machine learning, data engineering and applied analytics.

Core Modules: Statistics & Probability, Machine Learning, Data Mining, Data Engineering, Big Data Platforms, Data Visualization, Python & R for Data Science.

Duration: 2-3 years (depending on diploma level) with short professional certificates available.

Career Paths: Data Scientist, Machine Learning Engineer, Data Analyst, BI Developer, Data Engineer.

Enquire / Apply

Robotics

Overview: Hands-on program combining mechanical systems, electronics, control systems and AI. Students design and program robots for industrial, research and consumer applications.

Core Modules: Embedded Systems, Control Theory, Sensors & Actuators, Mechatronics, Robot Kinematics, Computer Vision, Reinforcement Learning.

Duration: 3 years (full-time) with lab-based projects and capstone.

Career Paths: Robotics Engineer, Automation Engineer, Researcher, Embedded Systems Developer, Autonomous Systems Engineer.
